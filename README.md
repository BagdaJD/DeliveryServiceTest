# DeliveryService 🚚

Сервис для расчета стоимости и прогнозирования времени доставки грузов. Проект разработан на языке Java с использованием современных практик тестирования (JUnit 5, JaCoCo).

---

## Технологический стек

* **Язык программирования:** Java 21
* **Сборщик проектов:** Maven
* **Тестирование:** JUnit 5 (Jupiter & Platform)
* **Анализ покрытия кода:** JaCoCo (версия 0.8.11+)

---

## Структура проекта и ключевые классы

* **`DeliveryService`** — основной класс бизнес-логики, содержащий методы:
  * `calculateCost(double distance, boolean isLarge, boolean isFragile, Status demand)` — расчет стоимости с учетом ограничений на хрупкость, габариты и текущий спрос.
  * `calculateTime(LocalDate startDate, double distance, boolean isFragile, Status demand)` — расчет даты прибытия с автоматическим переносом выходных дней (воскресенья) на понедельник.
* **`DeliveryService.Status`** — перечисление (Enum) для отображения уровня спроса (`NORMAL`, `HIGH` и т.д.).
* **`DeliveryServiceTest`** — комплексный тестовый класс, включающий в себя:
  * Юнит-тесты на граничные значения и бизнес-логику (`@Nested` блоки).
  * Тестирование обработки исключений (`assertThrows`, `assertAll`).
  * Параметризованные динамические тесты (`@TestFactory`, `DynamicTest`).
  * Тегирование тестов для регрессионного запуска (`@Tag`).
